# Кордон

Настольная утилита СКУД для ParsecNET: выбор группы приборов (ГП), подмена
`parsec.ini`, запуск ParsecNET и очистка конфига при выходе.

Версия 2.0 переписана с Electron на **Tauri v2** (React + TypeScript, лёгкий
нативный бинарь под Windows) с обновлённым дизайном и экраном настроек.

## Возможности

- Сетка кнопок ГП с явной подсветкой применённого профиля и статус-строкой.
- «Пуск» запускает ParsecNET, «Закрыть» удаляет применённый `parsec.ini` и выходит.
- Экран настроек: список профилей (добавить/удалить/переименовать), пути к
  программе, целевому `parsec.ini` и папке профилей GP, выбор темы
  (светлая / тёмная / системная).
- Настройки хранятся в `%APPDATA%/com.kordon.app/settings.json`.

## Как это работает

Каждый профиль — папка внутри «папки профилей GP» с файлом `parsec.ini`.
По клику `<gpFolder>/<folder>/parsec.ini` копируется в целевой путь. Относительная
папка GP (по умолчанию `GP`) резолвится рядом с исполняемым файлом — портативная
раскладка «папка GP рядом с .exe» работает из коробки.

## Разработка

Требуется Node.js 18+ и Rust (stable) с [пререквизитами Tauri](https://v2.tauri.app/start/prerequisites/).

```bash
npm install
npm run tauri dev    # полноценное приложение с hot-reload
```

Только для просмотра дизайна (без Rust/сборки) можно открыть фронтенд в браузере:

```bash
npm run dev          # http://localhost:1420
```

Вне окна Tauri вызовы к бэкенду подменяются мок-данными (настройки хранятся в
`localStorage`), поэтому оба экрана — главный и настройки — листаются и в браузере.

## Сборка под Windows

Собирать нужно на Windows. На Windows-машине:

```bash
npm install
npm run tauri build
```

Готовые установщики появятся в `src-tauri/target/release/bundle/` (NSIS `.exe`
и `.msi`).

## Иконки

Иконки сгенерированы из `img/app-icon.png` командой `npx tauri icon`.
Чтобы заменить: положите квадратный PNG (не менее 512x512) и выполните
`npx tauri icon путь/к/icon.png`.
